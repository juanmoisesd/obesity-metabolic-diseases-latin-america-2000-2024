## Overview

Comprehensive English-language database on the obesity epidemic and metabolic disease burden across **14 Latin American countries** (2000–2024).

**Author:** Juan Moisés de la Serna Tuya  
**ORCID:** [0000-0002-8401-8018](https://orcid.org/0000-0002-8401-8018)  
**OSF Repository:** [https://osf.io/w2e8h/](https://osf.io/w2e8h/)

## Contents

Longitudinal data on:

1. **Adult obesity prevalence** (BMI≥30) — three time points (2000, 2010, 2022)
2. **Type 2 diabetes prevalence** (%)
3. **Hypertension prevalence** (≥140/90 mmHg)
4. **Metabolic syndrome rates** (ATP III criteria)
5. **Physical inactivity levels** (%)
6. **Caloric intake trends** (kcal/day changes 2000–2020)
7. **Annual trend series** 2000–2022

## Countries Covered (14)

Argentina, Brazil, Chile, Colombia, Mexico, Peru, Bolivia, Ecuador, Venezuela, Uruguay, Paraguay, Costa Rica, Guatemala, Honduras

## Key Findings

- Obesity rose **+69%** since 2000 in the region
- Diabetes increased **+85%** since 2000
- Rapid nutritional transition documented across all countries
